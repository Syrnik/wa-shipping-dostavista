# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.3.0]

### Added
- Поддержка расчёта same_day

## [2.2.0]

### Added
- Выбор типа транспорта (мотоцикл, автомобиль и др.)
- Поддержка страховки заказа: сумма без скидок, итоговая сумма или произвольная формула
- Ограничения по весу заказа: минимальный и максимальный допустимый вес

## [2.1.0]

### Added
- Поддержка наложенного платежа (cash on delivery)
- Кэширование ответов API Dostavista

## [2.0.0]

### Changed
- Переход на новый API-клиент Dostavista (REST v2)
- Рефакторинг: логирование через PSR-3 (`psr/log`, `serger/process-logger`)
- Наценки к тарифу вынесены в общую библиотеку `syrnik/wa-shipping-utils`
- Поддержка интерфейса Webasyst 2.0 (`settings.html`) и устаревшего (`settings-legacy.html`)

### Removed
- Удалены внутренние классы `LoggerForShippingPlugins`, `LoggerActionFormatter`, `Surcharge`,
  `InstanceCache`, `Address` — заменены вендорными библиотеками
- Удалена зависимость `webit` (вендор)

### Fixed
- Очистка остатков старого пакета `syrnik/wa-sipping-utils` (опечатка в названии) — 2024-01-22

## [1.1.0]

### Added
- Доставка день-в-день (`same_day`): автоматический выбор тарифа по интервалам API
- Возможность покупателю выбрать желаемую дату и интервал доставки
- Настройка праздничных и дополнительных рабочих дней

## [1.0.0]

### Added
- Расчёт стоимости курьерской доставки через API Dostavista
- Настройка операционного региона (один или несколько регионов России)
- Фильтрация географии по белому/чёрному списку городов
- Настройка даты отправки (день в день, на следующий день и т.д.)
- Наценки к тарифу: абсолютные, процентные, формульные
- Отображение ориентировочной даты доставки в корзине
- Поддержка только рублей (RUB)
- Логирование запросов к API
