# Плагин расчёта стоимости доставки Dostavista для Webasyst Framework

Плагин позволяет рассчитать стоимость доставки курьерами службы Dostavista.

Расчёт выполняется для одиночной доставки от вашего магазина до адреса, указанного клиентом.

## География

В плагине есть возможность ограничить географию доставок по названию населённого пункта: либо только для указанного списка городов, либо наборот -- по всему региону, кроме указанного списка городов.

## Дата и время доставки

В плагине предусмотрена настройка даты отправки: когда вы готовы будете отправить заказ (в день получения, на след. день и т.д.), а также возможность настроить интервалы доставки, которые может выбрать покупатель при оформлении заказа.

### Доставка день-в-день

Если у вас настроена доставка «в тот же день», плагин посчитает тариф с опцией "same_day" (он обычно дороже доставки в последующие дни). Если у вас включена опция, позволяющая покупателю выбрать дату доставки и покупатель выберет дату в будущем — плагин пересчитает стоимость, без этой опции.

## Наценки к тарифу

К стоимости доставки, посчитанной сервером Dostavista, вы можете настроить различные наценки: в абсолютных значениях, в процентах и т.п.
